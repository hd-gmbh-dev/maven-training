# Exercise 2 — Adding a Dependency

The dependency was added in the [`pom.xml`](pom.xml) and the example code was added to [`Phase4Application.java`](src/main/java/com/example/Phase4Application.java).

Run the following commands to execute it (the jar name depends on your artifact id):

```shell
$ mvn package
$ java -jar target/exercise-02-dependency-solution-0.0.1-SNAPSHOT.jar
```

You should see the Spring header followed by your json.

To execute the Tests, simply run:

```shell
$ mvn test
```
